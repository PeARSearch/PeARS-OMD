PeARS-OMD: introduction

This PeARS version is dedicated to the On My Disk (OMD) project. On My Disk is a great solution for all privacy-conscious individuals who want to run their own private cloud. The software is completely plug-and-play, so you don't need to know anything about setting up a cloud. With the addition of PeARS, you can also search through your files as if you were your own search engine. The present Web app is a handy interface for searching your private cloud.


Configuring your PeARS installation

After installing PeARS, you will want to head to the configuration file situated at app/static/conf/pears.ini. The main thing you want to change there is the languages you will use in your installation. You should change the line
LANGUAGES=...
to show the languages you want to use as a comma-separated list. For instance, if you speak English and Slovenian:
LANGUAGES=en,sl


Signing in

To log in, you should use your On My Disk credentials. If you have lost your username or password, please get them back from your On My Disk account by going to this page.


Making your On My Disk content searchable

In order to search your content, you first need to 'index' it. This means that PeARS will generate mathematical representations of your documents that will be used at search time, like the big search engines. To do this, head over to your indexing page. Enter a directory to crawl from. Typically, if you want to index all your files, this will be a url of the type https://demo.onmydisk.net/your-username/.


Searching in several languages

Your PeARS installation contains pretrained models for four different languages: English, French, Russian and Slovenian. We will add more in the future! You don't have to do anything at indexing time. PeARS will automatically recognise the language of your document and use the right model to index it. At search time, the engine will search through all languages by default, but you can restrict this by using a language flag. For instance:
Paris hotel
would search through all documents that contain those two words. But
Paris hotel -en
would only return the documents in English, while
Paris hotel -fr
would only return the documents in French.
